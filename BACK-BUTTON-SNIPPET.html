← Portal Back-Button — Copy-Paste-Anleitung

Diese Datei ist nicht Teil der App. Sie ist nur deine Anleitung, wie du den Back-Button in die drei HTML-Apps einsetzt.

Wofür? Wenn der Nutzer aus dem Portal heraus eine App öffnet (z. B. KalkuPorte), braucht er einen Weg zurück zum Portal. Der Button setzt sich oben links als schwebender Pill ab — unauffällig, aber immer erreichbar. Kompatibel mit Splash-/Header-Layouts der bestehenden Apps.

Schritt 1 — Das Snippet

Kopiere den folgenden Block direkt nach dem öffnenden <body> Tag jeder App (also bei TiefbauPorte, KalkuPorte und GK-Rechner):

<!-- Genius Portal Back-Button -->
<a href="../index.html" id="gp-back" aria-label="Zurück zum Genius Portal">
  <svg viewBox="0 0 24 24" width="16" height="16" fill="none" stroke="currentColor" stroke-width="2.4" stroke-linecap="round" stroke-linejoin="round">
    <path d="M19 12H5M11 18l-6-6 6-6"/>
  </svg>
  <span>Portal</span>
</a>
<style>
  #gp-back {
    position: fixed;
    top: calc(env(safe-area-inset-top, 0px) + 10px);
    left: calc(env(safe-area-inset-left, 0px) + 10px);
    z-index: 10000;
    display: inline-flex;
    align-items: center;
    gap: 6px;
    padding: 8px 14px 8px 10px;
    background: rgba(27, 45, 94, 0.94);
    color: #fff;
    border-radius: 999px;
    font: 600 12px/1 "Calibri", "Inter", -apple-system, sans-serif;
    letter-spacing: .08em;
    text-transform: uppercase;
    text-decoration: none;
    box-shadow: 0 4px 14px rgba(15, 27, 58, .25);
    backdrop-filter: saturate(140%) blur(8px);
    -webkit-backdrop-filter: saturate(140%) blur(8px);
    transition: background .2s, transform .15s;
  }
  #gp-back:hover { background: #1B2D5E; transform: translateX(-1px); }
  #gp-back:active { transform: translateX(-2px) scale(.97); }
  #gp-back svg { flex: none; }
  /* Hide back button when the page is NOT inside the portal structure
     (e.g. opened standalone). Comment out these 3 lines to always show. */
  @media (display-mode: browser) {
    /* still show in browser */
  }
</style>

Schritt 2 — Wo genau einfügen?

  1. Öffne tiefbauporte.html in einem Text-Editor (VS Code / Notepad++).
  2. Suche nach <body — es gibt nur eine Stelle.
  3. Direkt nach dem schließenden > des <body>-Tags das Snippet einfügen.
  4. Speichern.
  5. Dasselbe für kalkuporte.html und gk-rechner.html.
  6. Alle drei Dateien in den Ordner apps/ des Portals packen.

Schritt 3 — Smoke-Test

  1. Öffne index.html im Browser.
  2. Tippe auf eine App-Kachel → die App öffnet sich mit dem "← Portal" Button oben links.
  3. Tippe auf "← Portal" → zurück im Dashboard.
Wenn eine App eine eigene Top-Navigation hat (z. B. TiefbauPorte mit dem Map-Header), kann der Button sich überlappen. In dem Fall: top im CSS erhöhen (z. B. auf 60px) oder den Button right statt left positionieren.

Alternative: Einmal einbauen, App bleibt unverändert

Wenn du deine Apps nicht anfassen willst: Portal-Link zurück ist auch per Browser-"Zurück"-Geste möglich (iOS: links reinwischen; Android: Back-Button). Der Back-Button ist komfortabler, aber kein Muss.
